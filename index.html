あなたの Web サイトです。

「iWeb」、「Pages」、その他のさまざまな Mac OS X 対応アプリケーションを使って、あなた独自の Web サイトを作成して公開しましょう。

Mac を使えば、Web サイトを作成して公開するのは簡単です。サイトの準備ができたら、さっそく公開してみましょう。

「システム環境設定」を開き、「共有」をクリックして、「Web 共有」を選択します。

これで完了です。あなたのサイトはこれで、ご家庭や職場のプライベートネットワーク上で見ることができます。

インターネットに接続している場合は、さまざまな場所にいる友人にアクセスしてもらうこともできます。「共有」環境設定に表示されるアドレスを友人に伝えるだけです。

Apache のパワー

「Web 共有」は、Mac OS X に含まれる業界標準のテクノロジー、Apache Web サーバを利用しています。Apache Web サーバについて詳しくは、Apache のマニュアルを参照してください。
